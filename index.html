Time tracking
that lives in Notion.

A lightweight Mac menu-bar app that logs billable hours directly to any Notion database — no extra tools, no subscriptions.

Download on the Mac App Store

Features

Logs to your own database

Works with any Notion database you already use. Map start time, end time, status, and project to whichever properties you choose.

Menu bar timer

A colored dot and live elapsed time appear in your menu bar while a timer is running — no window to keep open.

Project & tag tracking

Assign each entry to a project (via a relation or select property) and an optional tag, so your Notion views stay organized.

History feed

See your 50 most recent entries at a glance and edit title, project, time, or status inline without leaving the app.

Crash recovery

If you quit the app while a timer is running, it detects the in-progress entry on next launch and lets you stop it cleanly.

One-click re-timer

Restart any previous entry with the same title, project, and tag pre-filled — useful for recurring tasks.
